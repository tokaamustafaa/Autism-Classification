# Autism Classification

## Overview
This project builds Machine Learning and Deep Learning models to classify Autism Spectrum Disorder (ASD) based on behavioral and demographic data.

## Dataset
The dataset includes:
- Behavioral screening answers  
- Demographic information  
- Medical-related features  

**Target:** ASD (Yes / No)


## Preprocessing
- Handling missing values  
- Encoding categorical features  
- Feature scaling  
- Data cleaning  


## Models
**Machine Learning:**
- Logistic Regression (Accuracy:99%)
- Decision Tree (Accuracy:89%) 
- Random Forest (Accuracy:89%) 

**Deep Learning:**
- Neural Network (Dense layers)
- Accuracy:92%


## Evaluation
- Accuracy  
- Precision  
- Recall  
- F1-score  



## Tools
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib, Seaborn
